# Vendor Evaluation & Scoring Process

---

## 1. Evaluation Criteria

### **Technical Fit**
- Technology stack  
- Database compatibility  
- Integration capabilities  
- Performance & scalability  

### **Functional Fit**
- Coverage of subsystem requirements  
- Industry‑specific features for FMCG distribution  
- Reporting & analytics  

### **Usability**
- User interface  
- Workflow clarity  
- Customization flexibility  

### **Vendor Capability**
- Experience in FMCG distribution  
- References  
- Support quality  
- Implementation methodology  

### **Commercial Factors**
- Licensing model  
- Implementation cost  
- Maintenance & support fees  

---

## 2. Weighting Model
- Technical: 30%  
- Functional: 40%  
- Vendor capability: 15%  
- Commercial: 15%  

(Weights can be adjusted per organization.)

---

## 3. Scoring Method
- Each requirement scored 1–5  
- Weighted score calculated  
- Total score determines ranking  

---

## 4. Demo Evaluation
- Vendors receive predefined demo scenarios  
- Evaluation team scores each scenario  
- Strengths/weaknesses documented  

---

## 5. Risk Assessment
- Technical risks  
- Vendor dependency  
- Implementation risks  
- Cost risks  

---

## 6. Final Decision
- Top 2–3 vendors selected  
- Final comparison  
- Management approval
